# Node JS
это среда выполнения JavaScript, позволяющая использовать его на сервере (backend) для создания масштабируемых, высокопроизводительных веб-приложений и работы с файловой системой. Основные возможности включают создание API, работу с базами данных, реальное время (WebSocket), а также использование единого языка (JS/TS) для frontend и backend
